# Hyttetilbygg

Visualiseringsverktøy for å tegne et tilbygg på en hytte — i 2D og 3D.
Bygget med Next.js, Tailwind og React Three Fiber.

**Første gang du kjører:** se `KOM_I_GANG.md`.
**Kontekst for Claude Code:** se `CLAUDE.md`.

## Kort versjon

```
npm install
npm run dev
```

Åpne http://localhost:3000. I et annet terminalvindu i samme mappe:

```
claude
```

Alle mål på byggene ligger i `lib/bygg.ts`.
